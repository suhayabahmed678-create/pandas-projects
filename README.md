# 💸 Smart Expense Tracker — Student Edition

A simple yet powerful **pandas-based CLI expense tracker** designed for students.

Track your daily spending, analyze habits, and stay within budget — all from your terminal.

---

Features

Add new expenses

View all expenses in table format

Monthly summary (using pandas groupby)

Budget tracking system

Set monthly budget per category

Delete expenses by ID

Top 5 highest spending

CSV-based storage (no database needed)

---

## 🧠 Tech Stack

- Python 🐍
- pandas 📊
- CSV (data storage)
